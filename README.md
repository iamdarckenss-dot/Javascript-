# Introdução ao JavaScript

**JavaScript** ou **JS** é uma linguagem de programação usada para criar páginas web dinamicas, interativas e inteligentes.

També é possivel criar aplicações no servidor usando **node.js**

---

## Variáveis 
```javascript

let nome = "Ana";
const idade = 20;
var antigo = true;
```
## Tipos de dados


## Condições
``` javascript
if (idade>=18){
    console.log("maior de idade")
}else {
    console.log("menor de idade")
}
```
